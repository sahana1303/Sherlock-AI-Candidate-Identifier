# 🕵️ Sherlock AI Candidate Identifier

An AI-powered real-time candidate identification system built with **Python** and **Streamlit** to assist interview integrity. The application analyzes candidate information, participant data, confidence scores, transcript insights, skill extraction, and AI-based risk detection to identify the correct interview candidate.

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/sahana1303/Sherlock-AI-Candidate-Identifier.git
```

Go to the project folder:

```bash
cd Sherlock-AI-Candidate-Identifier
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

- Webcam Verification Status
- Confidence Score
- Risk Detection Score
