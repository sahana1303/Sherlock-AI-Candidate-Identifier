# Sherlock AI Candidate Identifier

## Overview

This project is an AI-powered candidate identification dashboard built for the Sherlock AI Internship Challenge.

It identifies the interview candidate using multiple signals such as:

- Candidate name
- Email matching
- Participant metadata
- Confidence scoring
- Explainable AI reasoning

## Features

- Live participant dashboard
- AI confidence scoring
- Resume match analysis
- Transcript analysis
- Skill extraction
- Risk detection
- Detection timeline
- Final AI decision
- Downloadable report

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
